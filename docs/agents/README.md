# AI 작업 지침

이 디렉터리는 Loki 문서를 다룰 때 사용하는 AI 작업 지침을 모은다.

## 문서 목록

| 문서 | 용도 |
|------|------|
| `doc-writer.md` | 문서 작성 기준 |
| `label-designer.md` | 레이블 설계 기준 |
| `logql-advisor.md` | LogQL 쿼리와 패턴 |
| `troubleshooter.md` | 문제 해결 절차 |

## 언제 보나

- 새 문서를 쓸 때 `doc-writer.md`
- 레이블을 설계할 때 `label-designer.md`
- LogQL 쿼리를 정리할 때 `logql-advisor.md`
- 장애 분석이나 재현이 필요할 때 `troubleshooter.md`
